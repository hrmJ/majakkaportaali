▸ SetEditTarget(level, force_all_levels, contentparent, string, boolean, string)

Asettaa sen, miten laajasti käyttäjä on muokkaamassa ulkoasua. Oletuksena koko esityksen laajuisesti, mutta valinnan mukaan myös joko segmenttityyppi- tai segmenttikohtaisesti. Käytännössä etsii kaikki ne kohteet, joita muutos koskee ja palauttaa yksittäisille muuttajafunktioille listan näistä kohteista.

SetEditTarget(level: any, force_all_levels: any, contentparent: any, string: any, boolean: any, string: any): any
Parameters
level (any)
force_all_levels (any)
contentparent (any)
string (any) level sen tekstitason (h1,h2,h3,p,body) nimi, jota muokkaus koskee
boolean (any) force_all_levels jos pakotetaan muokkaamaan jokaista tekstitasoa erikseen (tämä on välttämätöntä, jos haluaa esim. muokata kerralla kaikkien tekstielementtien väriä)
string (any) contentparent Määritelläänkö tyylimuutos erikseen sisällön taustalla olevalle elementille eikä koko ruudulle. Jos määritellään, arvoksi annetaan <välilyonti> + article
Returns
any: Array lista säännöistä, joita muutos koskee
